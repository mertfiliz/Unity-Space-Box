# Unity-Space-Box

(2020)

This is a 2D Casual mobile game where we control a box. The goal is to reach new planets by avoiding traps and reaching new highscores. It's developed by using Unity 2D with C#.

The game is released on Google Play Store under the company named of 'FrostyLeaf' (Description and images are included)

Here is the link: https://play.google.com/store/apps/details?id=com.FrostyLeaf.SpaceBox
